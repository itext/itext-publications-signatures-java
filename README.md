## License Requirement

A valid iText license is needed to run these examples. You can request a license here: [iText License Request](https://itextpdf.com/request-a-quote).

Before running the tests, set the `ITEXT_LICENSE_FILE_LOCAL_STORAGE` environment variable to point to your license file.
